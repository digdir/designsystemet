# Pagination

`Pagination` er en liste av knapper som ofte brukes for å navigere mellom sider av informasjon.

Komponenten kan brukes på forskjellige måter, se under [bruk](/docs/komponenter-pagination--docs#bruk) for mer informasjon

Vær oppmerksom på:

- Knappene er som standard `button` elementer.
- Dersom ny side fører til ny side sørg for at paginering knapper er `a` elementer.
  - Se eksempel [Paginering med lenker](/docs/komponenter-pagination--with-anchor).

## Bruk

### Eksempel på bruk med `Pagination`

`Pagination` er en kontrollert komponent. Det vil si at komponentens tilstand om hvilken side som er aktiv styres utenfra.

```tsx
import { Pagination } from '@digdir/designsystemet-react';

const [currentPage, setCurrentPage] = useState(4);
const { pages, goNext, goPrevious, hasNext, hasPrevious } = usePagination({
  currentPage,
  setCurrentPage,
  totalPages: 10,
  showPages: 7,
});

<Pagination>
  <Pagination.List>
    <Pagination.Item>
      <Pagination.Button
        aria-label='Forrige side'
        disabled={!hasPrevious}
        onClick={goPrevious}
      >
        Forrige
      </Pagination.Button>
    </Pagination.Item>
    {pages.map(({ page, itemKey, buttonProps }) => (
      <Pagination.Item key={itemKey}>
        <Pagination.Button {...buttonProps} aria-label={`Side ${page}`}>
          {page}
        </Pagination.Button>
      </Pagination.Item>
    ))}
    <Pagination.Item>
      <Pagination.Button
        aria-label='Neste side'
        disabled={!hasNext}
        onClick={goNext}
      >
        Neste
      </Pagination.Button>
    </Pagination.Item>
  </Pagination.List>
</Pagination>
```

## Paginering logikk

Bruk `usePagination` for paginering logikk sammen med de enkelt del-komponentene for mer avanserte og tilpasset paginering komponent.

`Pagination` er bygget på `usePagination`.

Bruk `key={itemKey}` på `Pagination.Item` og `...buttonProps` på `Pagination.Button` for å sikre universell utforming og indikere aktiv side.
