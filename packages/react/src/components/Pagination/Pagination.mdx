# Pagination

<Information text='development' />

`Pagination` er en liste av knapper som ofte brukes for å navigere mellom sider av informasjon. 

Dette er en controlled component som vil si at komponenten må ha en funksjon som setter hvilken side som er aktiv for å fungere. By default på hver item-button bortsett fra pilene er `aria-label=Page {sideNummer}` men dette kan overstyres ved å sette propen `itemLabel` til ønsket ord. Når komponenten er på første side skjules `previousLabel` og forrige-pilen. Når komponenten er på siste side skjules `nextLabel` og neste-pilen.

## Bruk

<Information text='token' />

```tsx
import '@digdir/design-system-tokens/brand/altinn/tokens.css'; // Importeres kun en gang i appen din.
import { Pagination } from '@digdir/design-system-react';

const [currentPage, setCurrentPage] = useState(1);

<Pagination
  currentPage={1}
  totalPages={10}
  onChange={setCurrentPage}
  nextLabel='Neste'
  previousLabel='Forrige'
  itemLabel={(num) => `Side ${num}}`
></Pagination>;
```
