<Controls />

## Slik bruker du `Tooltip`

Du kan enten sende inn en string eller et element som `children`. Dersom du sender inn et element, vil det elementet rendres i tooltipen.
Dersom du bruker kun en string vil det komme en `span` rundt teksten.

```tsx
import { Tooltip } from '@digdir/designsystemet-react';

// kun string, rendrer `span`
<Tooltip content="Vises i tooltip">
  Trigger
</Tooltip>

// med element, rendrer ditt element
<Tooltip content="Vises i tooltip">
  <MyButton>Trigger</MyButton>
</Tooltip>
```
