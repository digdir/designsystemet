# Chip

<Information text='beta' />

`Chips` er små avrundede knapper som kan brukes til å filtrere og navigere gjennom innhold eller data. For eksempel kan de brukes til å filtrere kategorier i et søkeresultat, samt vise hvilke filter som er aktive. Ikke bruk Chip til statisk metadata for å tagge innholdselementer. Bruk heller [Tag](https://digdir.github.io/designsystem/?path=/docs/kjernekomponenter-tag--docs) -komponenten dersom informasjonen ikke er klikkbar.

Chip eksisterer i ulike varianter:

- `Chip.Toggle` er et alternativ til `Button`. Den brukes når handlingen(e) er direkte relatert til hovedinnholdet og når handlingen ikke krever like mye oppmerksomhet som vanlige knapper. Det kan for eksempel være å klikke på en Chip for så å bli sendt til en søkeresultatside med filtrering på denne kategorien.
- `Chip.Toggle` med `selected=true` er alternativer til vanlige radiobuttons og checkboxer. Brukes til filtrering av innhold og data, når det skal tilbys et valg mellom flere ulike opsjoner. Innholdet endres til å vise kun det som er valgt.
- `Chip.Removable` er valgte verdier som brukeren kan fjerne. For eksempel filter som har blitt aktivert i et søk som kan fjernes igjen, eller som har blitt valgt i en `Multiselect`.
- `Chip` har samme bruksområde som `Chip.Toggle`, men native elementet er en `<a>` istedenfor `<button>`. Bruk denne dersom du ønsker å bruke Chip som en lenke. `Chip` støtter polymorfisme, og kan derfor rendres som andre elementer ved å bruke `as` egenskapen. For eksempel `<Chip as="span">`, men det kan være lurt å se om andre komponenter enn `Chip` passer bedre for ditt brukstilfelle.

## Bruk

<Information text='token' />

```tsx
import '@digdir/design-system-tokens/brand/altinn/tokens.css'; // Importeres kun en gang i appen din.
import { Chip } from '@digdir/design-system-react';

<Chip>You are using the Chip component!</Chip>;
```

Chip-komponenten består av flere komponenter. Disse er:

- `Chip`
- `Chip.Toggle`
- `Chip.Removable`
- `Chip.Group`

Som standard så rendres Chip som et`<a>` element, men dette kan overstyres ved å bruke `as` egenskapen, `<Chip as="span">`.Dette vil gjøre at en Chip rendres som en span istedenfor anchor tag.
Ønsker du å bruke Chip med `<button>` elementet kan du bruke `Chip.Toggle` istedenfor.
