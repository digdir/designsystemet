# Alert

<Information text='development' />

`Alert` skal brukes til å vise viktige meldinger, varsler eller advarsler til brukere. Den er designet for å fange brukernes oppmerksomhet og gi dem relevant informasjon som krever handling eller gjøre dem bevisst på viktig informasjon
