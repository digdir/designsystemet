# Link

`Link` kalles lenke på norsk. Det er en klikkbar tekst eller grafikk, 
som gjør det enkelt og raskt for brukerne å navigere til filer eller sider internt eller eksternt.
Komponenten er basert på HTML-taggen `<a>`.

<Primary />
<Controls />

## Slik bruker du `Link`

```tsx
import { Link } from '@digdir/designsystemet-react';

<Link href='https://designsystemet.no/'>Gå til designsystemet</Link>;
```

### I tekst

En lenke brukes ofte som en del av en tekst. Teksten som omfattes av lenken bør indikere tydelig hvor den går.
