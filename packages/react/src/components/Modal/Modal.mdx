# Modal

`Modal` bruker native `dialog` elementet, og for å åpne/lukke må du bruke `ref`.
Dette er valgt for å ha en så lett modal som mulig, og den vil derfor fungere annerledes enn våre andre komponenter.
Les [MDN Docs](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/dialog) for mer info.
