# Input

`Input` er et skjemaelement for å samle inn brukerdata. Det tilbyr grunnleggende funksjonalitet og er ideell når du trenger full kontroll over komponentenes oppsett og validering, noe som gjør det ideelt for bygging av spesialtilpassede elementer.

Se også [`Textfield`](docs/komponenter-textfield--docs) som er en gruppering av komponenter som ofte brukes sammen. Med `Textfield` slipper du å sy sammen `Label`, `Description`, `Input`, `ValidationMessage` selv hver gang.

### Plassering av ledeteksten

Ledeteksten og eventuell beskrivelse skal alltid stå over tekstfeltet. Dette gjør dem lette å se på små skjermer og hindres ikke av eventuelle feilmeldinger.
