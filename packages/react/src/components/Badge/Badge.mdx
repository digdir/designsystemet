# Badge

## Bruk

```tsx
import { Badge } from '@digdir/designsystemet-react';

/* Inline */
<Badge count={5} />

/* Flytende over element */
<Badge count={5}>
  <Elem />
</Badge>

/* Flytende over element som dot */
<Badge>
  <Elem />
</Badge>
```
