# Badge

`Badge` er ein ikkje-interaktiv komponent som viser status med eller utan tal. Bruk [Tag](/docs/komponenter-tag--docs) dersom du skal
ha tekst i staden for tal.

<Primary />
<Controls />

## Bruk

```tsx
import { Badge } from '@digdir/designsystemet-react';

/* Inline */
<Badge />

/* Flytende over element */
<Badge.Position>
  <Badge />
  <Elem />
</Badge.Position>
```

## Kodeeksempel

### Flytande over element

Du kan setje plassering med `placement`. Dersom elementet du plasserer over er ein sirkel, kan du setje `overlap="circle"`.

<Canvas of={BadgeStories.Floating} />

Ikkje alle ikon har lik form, og då må du kanskje endre kor `Badge` skal plasserast. Dersom du ikkje får `Badge` til å liggje der du ynskjer, kan du bruke `style` for å overstyre plasseringa.
Dette fungerer også når du bruker `overlap="circle"`.

<Canvas of={BadgeStories.CustomPlacement} />

### Fargar

`Badge` finst i alle fargar. Du kan endre fargen med `data-color`. 
Du kan også setje `variant` direkte for å endre bakgrunnsfargen.

<Canvas of={BadgeStories.Variants} />

### Vise staus

Dersom du ikkje sender inn `count`, blir berre ein sirkel vist. Dette kan brukast for å vise status.

<Canvas of={BadgeStories.Status} />

### Inline

Du kan bruke komponenten anten inline eller flytande over eit element.

<Canvas of={BadgeStories.InTabs} />

## Retningslinjer
Bruk `Badge` for å trekkje merksemd mot statusar, varslar eller tal. 

**Passer til:**
- Tal som viser antall nye meldingar, varsel eller oppgåver.
- Små statusindikatorar i form av ein sirkel, som for eksempel viser om ein person er opptatt, borte eller aktiv.

**Passer ikkje til:**
- Tekstar – bruk `Tag` i staden.
- Interaktive handlingar - bruk `Chip` eller `Button` i staden.

## Tekst
Denne komponenten er ikkje meint for tekst. Viss du treng å vise ord eller korte frasar, bør du heller bruke `Tag`-komponenten.
