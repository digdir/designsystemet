### Anbefalinger

En eller flere tags kan brukes til å kategorisere elementer og på den måten forbedre oversikten. Bruk tags når innholdet har mange kategorier og det er hensiktsmessig å kunne skille mellom dem. Kan også brukes til å kommunisere fremdrift og status.
Bruk korte etiketter for enkel skanning. Helst ett ord. Maks to ord dersom det er nødvendig for å beskrive statusen og skille den fra andre tagger.

### Ikke egnet til

- Tag er ikke egnet til link til andre sider (bruk heller link).
- Tag er ikke egnet til handlinger (bruk heller button).
