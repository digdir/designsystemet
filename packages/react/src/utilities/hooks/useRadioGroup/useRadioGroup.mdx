# useRadioGroup

En react-hook for å lettere håndtere en gruppe med `Radio`.

<Primary />
<Controls />


## Bruk

Under er et enkelt eksempel på hvordan du bruker `useRadioGroup`-hooken for å håndtere en radio gruppe.

I `getRadioProps` denne funksjonen er første parameter `value` som er en unik identifikator for radioen, eller et objekt med alt som kan legges på en radio.
Dersom du sender inn eit objekt, så må du passe på å ha med `value`.

**Det er viktig at du sender inn props i `getRadioProps`, slik at du ikke overskriver vår funksjonalitet.**

```tsx
const { getRadioProps, validationMessageProps } = useRadioGroup({
  name: 'my-radio-group',
  value: 'sjokolade',
  error: 'Du må velge et alternativ',
});

return (
  <Fieldset>
    <Fieldset.Legend>Hvilken iskremsmak er best?</Fieldset.Legend>
    <Fieldset.Description>
      Velg din favorittsmak blant alternativene.
    </Fieldset.Description>
    <Radio label='Vanilje' {...getRadioProps('vanilje')} />
    <Radio label='Jordbær' {...getRadioProps('jordbær')} />
    <Radio label='Sjokolade' {...getRadioProps('sjokolade')} />
    <ValidationMessage {...validationMessageProps} />
  </Fieldset>
);
```
<br />
I respons får du objektet:

```ts
{
  getRadioProps: (value: string | RadioProps) => RadioProps, // Returnerer props du kan spre på <Radio /> eller <input type="radio" />
  validationMessageProps: ValidationMessageProps, // Props du valgfritt sprer på ValidationMessage for å koble feilmelding til radio gruppen
  setValue: string, // Funksjon for å sette radio gruppens verdi
  value: string, // Nåværende verdi for radio gruppen
}
```
