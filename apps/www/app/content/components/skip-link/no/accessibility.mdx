Skip link er vanlegvis ikkje synleg før eit trykk på tastaturet aktiverer han.

Ver merksam på at nokre automatiske testar for tilgjengelegheit kan varsle om at skip link manglar i eit landemerke, for eksempel `<nav>`. Dette varselet gjeld ikkje for skip link, så det kan du oversjå.  

Pass på at skip link oppfører seg føreseieleg. Ein skip link med teksten «Gå direkte til innhaldet» skal flytte brukarens fokus til starten på innhaldet, ikkje til starten av det første avsnittet under overskrifta.
