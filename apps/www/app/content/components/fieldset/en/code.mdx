It is common to get confused between `Fieldset` and `Field`.
A good rule of thumb is that **`Fieldset` is a set of `Field`**

## Usage

```tsx
import { Fieldset, Radio } from '@digdir/designsystemet-react';

<Fieldset>
<Fieldset.Legend>Question</Fieldset.Legend>
<Fieldset.Description>Description</Fieldset.Description>
<Radio label="Option 1" name="radio" value="1" />
<Radio label="Option 2" name="radio" value="2" />
</Fieldset>
```

### Legend as heading

If it is a page that only has form on it, and the title of the page (`<h1>`) is the same as what will be in `<legend>`, you can put `<h1>` inside `Fieldset.Legend`.

## HTML

The class name `ds-fieldset` is added to the `<fieldset>` element.

```html
<fieldset class="ds-fieldset">
<legend class="ds-label" data-weight="medium">Do you accept the terms?</legend>
<!-- your fields -->
</fieldset>
```
