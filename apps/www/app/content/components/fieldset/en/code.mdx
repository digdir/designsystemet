It is common to get confused between fieldset and field.
A good rule of thumb is that **fieldset is a set of fields**

<Story story="PreviewEn" defaultOpen />

## Usage
The class name `ds-fieldset` is added to the `<fieldset>` element.

```html
<fieldset class="ds-fieldset">
  <legend class="ds-label" data-weight="medium">Do you accept the terms?</legend>
  <!-- your fields -->
</fieldset>
```

The component will inherit from the closest parent where these are set.
