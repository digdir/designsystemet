## Bruk

```tsx
import { Tabs } from '@digdir/designsystemet-react';

<Tabs defaultValue="tab1">
  <Tabs.List>
    <Tabs.Tab value="tab1">Tab 1</Tabs.Tab>
    <Tabs.Tab value="tab2">Tab 2</Tabs.Tab>
  </Tabs.List>
  <Tabs.Panel value="tab1">Content 1</Tabs.Panel>
  <Tabs.Panel value="tab2">Content 2</Tabs.Panel>
</Tabs>
```

### HTML
Du kan bruke [u-tabs (u-elements.github.io)](https://u-elements.github.io/u-elements/elements/u-tabs), som er en web-komponent som implementerer tabs.
Legg p책 klassenavnet `ds-tabs`.

```html
<u-tabs class="ds-tabs">
  <u-tablist>
    <u-tab>Tab 1</u-tab>
    <u-tab>Tab 2</u-tab>
  </u-tablist>
  <u-tabpanel>Panel 1</u-tabpanel>
  <u-tabpanel>Panel 2</u-tabpanel>
</u-tabs>
```

Vi anbefaler 책 bruke `<u-tabs>`, og ikke bygge egne tabs fra bunnen av, for 책 sikre god tilgjengelighet og funksjonalitet.
