## Usage

Apply the class name `ds-tag` to the element that should be a tag.
You can set `variant="outline"` to add an outline to the tag.

## CSS variables and data attributes
Size is controlled with [data-size](https://www.designsystemet.no/en/fundamentals/code/sizes) and color with [data-color](https://www.designsystemet.no/en/fundamentals/code/colors). The component inherits these from the closest parent where they are set.

```tsx
import { Tag } from '@digdir/designsystemet-react';

<Tag>Beta</Tag>;
```
