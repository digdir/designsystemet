## Usage
Use the `@digdir/designsystemet-web` package to implement `Tooltip`.
You only need to add the `data-tooltip` attribute to the element that will trigger the tooltip.
For accessibility, the element will automatically receive either `aria-label` or `aria-description`, depending on whether it contains text content.

```html
<button data-placement="left" data-tooltip="left" class="ds-button">left</button>
```

## Custom element
Use the `setTooltipElement` function to set a custom tooltip element.
The element will auto automatically be added do `document.body` if it is not already in the document.

```js
import { setTooltipElement } from @digdir/designsystemet-web

const myTooltip = document.createElement('div');
myTooltip.className = 'ds-tooltip my-custom-tooltip';
setTooltipElement(myTooltip);
```

## Text from another HTML element
You can use the `data-tooltip` attribute with a CSS ID selector to point to another element that contains tooltip text.

```html
<button data-tooltip="#tooltip-content" class="ds-button">venstre</button>
<div id="tooltip-content" class="ds-sr-only">Dette er innholdet i tooltipen.</div>
```

## CSS variables and data attributes
Sizes are controlled with [data-size](/en/fundamentals/code/sizes) and colors with [data-color](/en/fundamentals/code/colors).
