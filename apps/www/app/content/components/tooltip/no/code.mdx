## Bruk
Bruk pakken `@digdir/designsystemet-web` for å implementere `Tooltip`.
Du trenger kun å legge på `data-tooltip`-attributtet på elementet som skal trigge tooltipen.
For tilgjengelighet vil elementet automatisk få `aria-label` eller `aria-description`, avhengig av om det inneholder tekstinnhold.

```html
<button data-placement="left" data-tooltip="venstre" class="ds-button">venstre</button>
```

## Egendefinert element
Bruk funksjonen `setTooltipElement` for å sette et egendefinert tooltip-element.
Elementet vil automatisk bli lagt til i `document.body` hvis det ikke allerede er i dokumentet.

```js
import { setTooltipElement } from @digdir/designsystemet-web

const myTooltip = document.createElement('div');
myTooltip.className = 'ds-tooltip my-custom-tooltip';
setTooltipElement(myTooltip);
```

## Tekst fra et annet HTML-element
Du kan bruke `data-tooltip`-attributtet med en CSS-ID-selector, for å peke til et annet element som inneholder tooltip-tekst.

```html
<button data-tooltip="#tooltip-content" class="ds-button">Knapp</button>
<div id="tooltip-content" class="ds-sr-only">Dette er innholdet i tooltipen.</div>
```
