## Bruk

Bruk klassenavnet `ds-input` på `<input>` sammen med `type="checkbox"` og `role="switch"`.

```html
<input
  class="ds-input"
  role="switch"
  type="checkbox"
/>
```

For et gyldig skjemaelement anbefaler vi å bruke `<ds-field>` sammen med en `<label>`.

```html
<ds-field>
  <label class="ds-label" >Switch</label>
  <input
    class="ds-input"
    role="switch"
    type="checkbox">
</ds-field>
```

## CSS variabler og data-attributter
Størrelser styres med [data-size](/no/fundamentals/code/sizes) og farger med [data-color](/no/fundamentals/code/colors). 
Komponenten vil arve nærmeste forelder med disse satt.

Switch er `ds-input` med `type="checkbox"` så variabler og data-attributter som gjelder for [input](/no/components/docs/input/overview) gjelder også for switch.

## React

I React er det enklere å bygge en switch, fordi vi har en ferdiglaget komponent som håndterer det meste av logikken og tilstandshåndteringen for oss.

`Switch` er en sammensatt komponent som bruker [field](/no/components/docs/field/overview), [label](/no/components/docs/label/overview), og [input](/no/components/docs/input/overview) for å lage en switch.


```tsx
import { Switch } from '@digdir/designsystemet-react';

<Switch label="Mørk modus" value="dark" />
```
