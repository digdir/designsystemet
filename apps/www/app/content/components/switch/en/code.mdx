## Usage

The class name `ds-input` is added to `<input>` along with `type="checkbox"` and `role="switch"`.

```html
<input
  class="ds-input"
  role="switch"
  type="checkbox"
/>
```

To mirror what the composite component in React does, you need to use `ds-field` together with a `<label>`.

To right-align `<input>`, add `data-position="end"` to `ds-field`.

```html
<div class="ds-field" data-position="end">
  <input
    class="ds-input"
    role="switch"
    type="checkbox"
    id="my-switch"
  >
  <label
    class="ds-label"
    data-weight="regular"
    for="my-switch"
  >Flightmode</label>
</div>
```

Sizes are controlled with [data-size](/en/fundamentals/code/sizes) and colors with [data-color](/en/fundamentals/code/colors). 
The component will inherit from the closest parent where these are set.
