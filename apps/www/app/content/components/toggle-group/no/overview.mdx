`ToggleGroup` er en gruppe av knapper hvor brukeren kan velge ett alternativ.

<Story story="Preview" />

## Bruk

```tsx
import { ToggleGroup } from '@digdir/designsystemet-react';

<ToggleGroup defaultValue="option1">
  <ToggleGroup.Item value="option1">Alternativ 1</ToggleGroup.Item>
  <ToggleGroup.Item value="option2">Alternativ 2</ToggleGroup.Item>
</ToggleGroup>
```
