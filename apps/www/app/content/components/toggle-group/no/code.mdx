## Bruk
Bruker du `@digdir/designsystemet-web` kan du implementere toggle group ved å bruke `data-toggle-group`-attributtet.

```html
<fieldset class="ds-toggle-group" data-toggle-group="Tekstjustering" data-variant="secondary">
  <label class="ds-button">
    <input type="radio" name="alignment-two" value="left" checked />
    Venstestilt
  </label>
  <label class="ds-button">
    <input type="radio" name="alignment-two" value="center" />
    Midtstilt
  </label>
  <label class="ds-button">
    <input type="radio" name="alignment-two" value="right" />
    Høyrestilt
  </label>
</fieldset>
```

### Gammel måte med `<button>`
**Her må du håndtere tastaturnavigasjon og ARIA-roller sjølv.**

Du kan lese om kva du må gjere for å implementere dette i [tilgjengelighetsdokumentasjonen](/no/components/docs/toggle-group/accessibility).

Klassenavnet `ds-toggle-group` blir lagt på ein `<div>` med `role="radiogroup"`.

Kvar knapp i gruppa er ein `<button>` med klassenavn `ds-button` med `role="radio"` og `aria-checked` som indikerer om knappen er vald eller ikkje.

```html
<div class="ds-toggle-group" role="radiogroup" data-variant="primary">
  <button
    class="ds-button"
    type="button"
    id="togglegroup-item-1"
    aria-checked="true"
    role="radio"
  >Innboks</button>
  <button
    class="ds-button"
    type="button"
    id="togglegroup-item-2"
    aria-checked="false"
    role="radio"
  >Sendt</button>
</div>
```
