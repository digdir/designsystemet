## Usage

`FileUpload` is a compound component made up of several parts:

- `FileUpload` – the upload area itself (a `<div>` with the class `ds-file-upload`).
- `FileUpload.Description` – secondary text inside the area.
- `FileUpload.FakeButton` – a visual button for a clear click target (rendered as a `Button`).
- `FileUpload.Input` – the hidden `<input type="file">`. Accepts all native input attributes such as `accept`, `multiple`, `disabled`, `readOnly` and `capture`.

Place `FileUpload` inside a `Field` with a `Label` so the upload area gets an accessible label.

### Plain HTML

In plain HTML, use the class `ds-file-upload` on an element that contains an `<input type="file">`. The input is placed on top of the whole area so the entire surface becomes clickable.

```html
<div class="ds-field">
  <label class="ds-label">Upload file</label>
  <div class="ds-file-upload">
    <p class="ds-paragraph" data-field="description">Drop file here</p>
    <input type="file" title="" />
  </div>
</div>
```

To make the area read-only, set `readonly` on the input. The component then automatically switches to a neutral color scale and disables interaction.

## React

```tsx
import { EXPERIMENTAL_FileUpload as FileUpload, Field, Label } from '@digdir/designsystemet-react';

<Field>
  <Label>Upload file</Label>
  <FileUpload>
    <FileUpload.Description>Drop file here</FileUpload.Description>
    <FileUpload.Description>
      File must be in csv format and less than 2MB
    </FileUpload.Description>
    <FileUpload.FakeButton>Upload file</FileUpload.FakeButton>
    <FileUpload.Input accept=".csv" />
  </FileUpload>
</Field>
```

### Using react-dropzone
For more advanced handling, `FileUpload` can be combined with [react-dropzone](https://react-dropzone.js.org/). Note that react-dropzone sets `tabindex="-1"` on the input and `tabindex="0"` on the element that receives `{...getRootProps()}`.
