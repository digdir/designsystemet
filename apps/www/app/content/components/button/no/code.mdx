## Bruk

```tsx
import { Button } from '@digdir/designsystemet-react';

<Button>Knapp</Button>;
```

## Eksempel

### Varianter

`variant` brukast for å endre utsjånad på knappen.

<Story story="Variants" />

### Fargar

Enten sett du `data-color` direkte på `Button`, eller så arvar den frå næraste overordna element med `data-color`.

<Story story="ColorVariants" />

### Ikon

Dersom du har kun ikon i knappen kan du sette `icon={true}` for å få den firkanta.
Har du anna innhald, som tekst, vil knappen automatisk få luft rundt ikonet.

<Story story="Icons" />

### Knapp som lenkje

`asChild` brukast for å endre knappen til ei lenkje.
Les meir kom [komposisjon i Designsystemet](/no/fundamentals/code/composition).

<Story story="AsLink" />

### Knappar som lastar

Under brukar me `loading={true}` for å vise at knappen lastar noko. Her kan du også sende inn din eigen loading-komponent dersom vår [`Spinner`](/no/components/spinner/code) ikkje passar.

<Story story="Loading" />

I eksempelet over blir `aria-disabled` sett automatisk når `loading={true}`. Dette er den anbefalte måten å vise at ein knapp er deaktivert på.
Då kan knappen framleis få fokus når brukaren navigerer med tastatur.
Slik blir skjermlesarar og andre hjelpemiddel informerte om at knappen finst, men at han ikkje er aktiv.

**Merk:** Verken `loading="true"` eller `aria-disabled` hindrar automatisk at knappen utløyser `onClick`.  
Du må sjølv sørgje for at callback-funksjonen ikkje køyrer når knappen er deaktivert.

## HTML

Klassenavnet `ds-button` blir satt på `<button>`.
I React legg me også på `type="button"` som standard, for å unngå uventa submit i skjema.

Det er tre data-attributtar du kan bruke for tilspasning:
`data-icon` - legg på dersom knappen kun har ikon.
`data-variant` - set variant på knappen (`primary`, `secondary`, `tertiary`).
`data-fullWidth` - legg på dersom knappen skal vere full bredde.

```html
<button class="ds-button"type="button">Knapp</button>

<!-- sekundær variant -->
<button class="ds-button" type="button" data-variant="secondary">Knapp</button>

<!-- kun ikon -->
<button class="ds-button" type="button" data-icon="true"><svg /></button>
```

Dersom du skal ha ein knapp som laster, må du sjølv legge til laste-indikator.

```html
<button class="ds-button" type="button" aria-disabled="true" aria-busy="true">
  <svg class="ds-spinner" />
  Lastar...
</button>
```
