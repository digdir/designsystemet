## Bruk

Bruk attributtet `data-color="info | success | warning | danger"` for å endre type varsel. 

Bruk `role="alert"` for kritiske meldinger som krever umiddelbar oppmerksomhet.
Bruk `role="status"` for mindre kritiske meldinger.
