## Guidelines

Use `Checkbox` when the user can select one or more options.

**Suitable for**

- selecting multiple options in forms
- accepting terms and conditions

**Not suitable for**

- selecting only one option, use [`Radio`](/docs/komponenter-radio--docs) instead
