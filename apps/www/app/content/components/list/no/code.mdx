## HTML

Du bruker enten `<ul>` eller `<ol>` med klassenavnet `ds-list`.
`<li>` brukes for hvert listepunkt, og trenger ingen klassenavn.

```html
<!-- usortert liste -->
<ul class="ds-list">
  <li>Item</li>
  <li>Item</li>
</ul>

<!-- sortert liste -->
<ol class="ds-list">
  <li>Item</li>
  <li>Item</li>
</ol>
```
