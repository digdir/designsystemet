## HTML

You use either `<ul>` or `<ol>` with the class name `ds-list`.
`<li>` is used for each list item, and does not need a class name.

```html
<!-- unsorted list -->
<ul class="ds-list">
<li>Item</li>
<li>Item</li>
</ul>

<!-- sorted list -->
<ol class="ds-list">
<li>Item</li>
<li>Item</li>
</ol>
```
