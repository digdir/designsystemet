`Table` is used to display structured data in rows and columns.

<Story story="Preview" />

## Usage

```tsx
import { Table } from '@digdir/designsystemet-react';

<Table>
  <Table.Head>
    <Table.Row>
      <Table.HeaderCell>Header</Table.HeaderCell>
    </Table.Row>
  </Table.Head>
  <Table.Body>
    <Table.Row>
      <Table.Cell>Cell</Table.Cell>
    </Table.Row>
  </Table.Body>
</Table>
```
