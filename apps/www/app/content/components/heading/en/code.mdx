## HTML

Use the class name `ds-heading` to style an HTML `<h1>`-`<h6>` element.

```html
<h1 class="ds-heading" data-size="xl">Heading level 1</h1>
<h2 class="ds-heading" data-size="lg">Heading level 2</h2>
```
