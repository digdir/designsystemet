## Usage

Use the class name `ds-search` on a wrapper element around `<input>`.
If you want a clear button, add a `<button>` with `type="reset"`.
Add an empty `placeholder` to `<input>` if you have a clear button.

## CSS variables and data attributes

Sizes are controlled with [data-size](/en/fundamentals/code/sizes) and colors with [data-color](/en/fundamentals/code/colors). The component will inherit from the closest parent where these are set.
