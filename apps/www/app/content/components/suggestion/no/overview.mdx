## Bruk

```tsx
import { EXPERIMENTAL_Suggestion as Suggestion } from '@digdir/designsystemet-react'

<Suggestion>
  <Suggestion.Input />
  <Suggestion.Clear />
  <Suggestion.List>
    <Suggestion.Empty>Tomt</Suggestion.Empty>
    <Suggestion.Option label="Sogndal" value="sogndal">
      Sogndal
    </Suggestion.Option>
    <Suggestion.Option label="Oslo" value="oslo">
      Oslo
    </Suggestion.Option>
  </Suggestion.List>
</Suggestion>
```

## Eksempel

### Flervalg
Du kan gi brukeren mulighet til å velge flere alternativer.
Slettbare `Chip`s vil komme etter `<label>`, men før inndatafeltet.

<Story story='Multiple' />

### Filter
Filter er som standard på, og filteret ser etter etter teksten i input.
Du kan sende inn ditt eget filkter.
Under viser vi et eksempel som skrur av filteret helt.

<Story story='Filter' />

## Tilleggsinformasjon
`Suggestion` baserer seg på [u-combobox fra u-elements (github.io)](https://u-elements.github.io/u-elements/elements/u-combobox). 
Den er inspisert av [open-ui sitt combobox mønster (open-ui.org)](https://open-ui.org/components/combobox.explainer/).
