## Bruk
Suggestion baserer seg på [open-ui sitt combobox mønster (open-ui.org)](https://open-ui.org/components/combobox.explainer/) og bruker [u-combobox fra u-elements (github.io)](https://u-elements.github.io/u-elements/elements/u-combobox) for å oppfylle grunnleggende funksjonalitet fra combobox mønsteret.

Bruk klassen `ds-suggestion` på custom elementet `<ds-suggestion>`.
Legg til `<input class="ds-input" />` og `<u-datalist>` med `<u-option>` som barn for å få en fungerende suggestion.

```html
<ds-suggestion class="ds-suggestion">
  <input class="ds-input"  />
  <u-datalist>
    <u-option label="Sogndal" value="Sogndal">Sogndal</u-option>
    <u-option label="Oslo" value="Oslo">Oslo</u-option>
    <u-option label="Brønnøysund" value="Brønnøysund">Brønnøysund</u-option>
  </u-datalist>
</ds-suggestion>
```

For et gyldig skjemaelement anbefaler vi å bruke `<ds-suggestion>` i `<ds-field>` sammen med en `<label>`.

```html
<ds-field>
  <label class="ds-label" >Checkbox</label>
  <ds-suggestion class="ds-suggestion">
    <input class="ds-input"  />
    <button type="reset" aria-label="Tøm"/>
    <u-datalist>
      <u-option label="Sogndal" value="Sogndal">Sogndal</u-option>
      <u-option label="Oslo" value="Oslo">Oslo</u-option>
      <u-option label="Brønnøysund" value="Brønnøysund">Brønnøysund</u-option>
    </u-datalist>
  </ds-suggestion>
</ds-field>
```

### Flervalg
For å tillate at brukeren kan velge flere alternativer, bruk `data-multiple` på suggestion.

<Story story='Multiple' />

### Legge til nye alternativer

Med `data-creatable` på suggestion kan en bruker opprette nye valgte alternativer ved å skrive inn tekst i input og trykke <kbd>Enter</kbd>.

<Story story='Creatable' />

### Tøm valget

Legg til en knapp med `type="reset"` og `aria-label` for å la brukeren tømme valget.

<Story story='Preview' />

### Åpne og lukk liste

Legg til en knapp med `type="button" aria-expanded="false"` og `aria-label` for å la brukeren åpne og lukke listen.
