## Bruk
`Suggestion` baserer seg på [u-combobox fra u-elements (github.io)](https://u-elements.github.io/u-elements/elements/u-combobox). Den er inspisert av [open-ui sitt combobox mønster (open-ui.org)](https://open-ui.org/components/combobox.explainer/).

```tsx
import { EXPERIMENTAL_Suggestion as Suggestion } from '@digdir/designsystemet-react'

<Suggestion>
  <Suggestion.Input />
  <Suggestion.Clear />
  <Suggestion.List>
    <Suggestion.Empty>Tomt</Suggestion.Empty>
    <Suggestion.Option label="Sogndal" value="sogndal">
      Sogndal
    </Suggestion.Option>
    <Suggestion.Option label="Oslo" value="oslo">
      Oslo
    </Suggestion.Option>
  </Suggestion.List>
</Suggestion>
```

## HTML
Vi utvider u-elements sine elementer.

Du finner generell dokumentasjon på bruk av `ds-combobox` hos u-elements:
[https://u-elements.github.io/u-elements/elements/u-combobox](https://u-elements.github.io/u-elements/elements/u-combobox).

For å få Designsystemet styling, legger du klassenavnet `ds-suggestion` på `<ds-combobox>` elementet.
`<input>` skal ha `ds-input` klassen.

Dersom du skrur på flervalg, skal `<data>` ha klassenavnet `ds-chip` og data attributtet `data-removable="true"`.

```html
<ds-combobox class="ds-suggestion" ...>
  <data class="ds-chip" data-removable="true" ...>Valgt alternativ</data>
  <input class="ds-input" ... />
  <ds-datalist>
    <ds-option label="Sogndal" value="sogndal">Sogndal</ds-option>
    <ds-option label="Oslo" value="oslo">Oslo</ds-option>
  </ds-datalist>
</ds-combobox>
```
