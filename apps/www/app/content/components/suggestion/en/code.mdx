## Usage
`Suggestion` is based on [u-combobox from u-elements (github.io)](https://u-elements.github.io/u-elements/elements/u-combobox). It is inspected by [open-ui's combobox pattern (open-ui.org)](https://open-ui.org/components/combobox.explainer/).

```tsx
import { EXPERIMENTAL_Suggestion as Suggestion } from '@digdir/designsystemet-react'

<Suggestion>
  <Suggestion.Input />
  <Suggestion.Clear />
  <Suggestion.List>
    <Suggestion.Empty>Empty</Suggestion.Empty>
    <Suggestion.Option label="Sogndal" value="sogndal">
    Sogndal
    </Suggestion.Option>
    <Suggestion.Option label="Oslo" value="oslo">
    Oslo
    </Suggestion.Option>
  </Suggestion.List>
</Suggestion>
```

## HTML
`ds-suggestion` is an extension of `u-combobox` from u-elements.
You can find general documentation on using `u-combobox` at u-elements:
[https://u-elements.github.io/u-elements/elements/u-combobox](https://u-elements.github.io/u-elements/elements/u-combobox).

To get the Design System styling, add the class name `ds-suggestion` to the `<ds-combobox>` element.
`<input>` should have the `ds-input` class.

If you enable multiple choice, `<data>` should have the class name `ds-chip` and the data attribute `data-removable="true"`.

```html
<ds-suggestion class="ds-suggestion" ...>
  <data class="ds-chip" data-removable="true" ...>Selected option</data>
  <input class="ds-input" ... />
  <ds-datalist>
    <ds-option label="Sogndal" value="sogndal">Sogndal</ds-option>
    <ds-option label="Oslo" value="oslo">Oslo</ds-option>
  </ds-datalist>
</ds-suggestion>
```
