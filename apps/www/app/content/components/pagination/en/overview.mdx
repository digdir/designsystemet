## Usage

`Pagination` consists of several sub-components:

- `Pagination` - Wrapper for the entire component
- `Pagination.List` - List of page buttons
- `Pagination.Item` - An item in the list (can be empty to show "...")
- `Pagination.Button` - Button to navigate to a page

## Guidelines

- Use Pagination when content is divided across multiple pages
- Always show which page the user is on
- Include "Previous" and "Next" buttons
- Use `aria-label` to provide context for screen readers
- Consider using a "Load more" button for smaller datasets

## usePagination hook

Use the `usePagination` hook to handle pagination logic:

```tsx
const { pages, nextButtonProps, prevButtonProps } = usePagination({
  currentPage: 1,
  setCurrentPage: (page) => console.log(page),
  totalPages: 10,
  showPages: 7,
});
```
