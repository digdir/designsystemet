## Bruk
`Pagination` er en komponent uten state, og du må selv styre hvilke sider som er vist.

Bruk `usePagination` for paginering-logikk sammen med de enkelt del-komponentene.
[Les mer om `usePagination`](/no/components/utilities/use-pagination).

```tsx
import { Pagination } from '@digdir/designsystemet-react';

<Pagination>
  <Pagination.List>
    <Pagination.Item>
      <Pagination.Button aria-label='Forrige side'>
        Forrige
      </Pagination.Button>
    </Pagination.Item>

    <Pagination.Item>
      <Pagination.Button aria-label='Side 1'>
        1
      </Pagination.Button>
    </Pagination.Item>

    <Pagination.Item />

    <Pagination.Item>
      <Pagination.Button aria-label='Side 10'>
        10
      </Pagination.Button>
    </Pagination.Item>

    <Pagination.Item>
      <Pagination.Button aria-label='Neste side'>
        Neste
      </Pagination.Button>
    </Pagination.Item>
  </Pagination.List>
</Pagination>
```
## Eksempler

### Standard

Eksempelet viser `Pagination` med knapper som ikke er koblet til noen handling.
Når du trykker vil du ikke navigere til en annen side.

<Story story="Preview" />

### Med lenker

Under ser du `usePagination` brukt sammen med `Pagination` for å lage paginering med lenker.
Du bruker `asChild` for å endre `Pagination.Button` til `<a>`.

<Story story="WithAnchor" />

### Mobil

Det er ikke noe innebygget mobilstøtte i `Pagination`, men du kan vise færre sider og fjerne tekst for forrige/neste knapper på mobile enheter.

## HTML

Legg klassenavnet `ds-pagination` på `<ds-pagination>` elementet.
Web komponenten fyller automatisk knappene med tekst basert på attributtene `data-current` og `data-total`.

Dersom du ønsker å lage dine egne knapper kan du legge inn `<a>` eller `<button>` elementer inni `<ds-pagination>`. Dersom du fyller disse med tekst, og ikke sender noen attributter, vil ikke web komponenten endre på knappene.

```html
<ds-pagination class="ds-pagination" aria-label="Bla i sider:" data-href="?page=%d" data-current="2" data-total="100">
  <ol>
    <li><a></a></li>
    <li><a></a></li>
    <li><a></a></li>
    <li><a></a></li>
  </ol>
</ds-pagination>
```
