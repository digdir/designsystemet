## HTML
Append the class name `ds-spinner` to a `<svg>` tag to create a spinner.
The `<svg>` should have two `<circle>` elements inside it to create the spinner.

Check the code snippet below for the complete HTML structure.

```html
<svg
  aria-label="Loading coffee"
  class="ds-spinner"
  role="img"
  viewBox="0 0 50 50"
>
  <circle
    class="ds-spinner__background"
    cx="25"
    cy="25"
    r="20"
    fill="none"
    stroke-width="5"
  ></circle>
  <circle
    class="ds-spinner__circle"
    cx="25"
    cy="25"
    r="20"
    fill="none"
    stroke-width="5"
  ></circle>
</svg>
```
