## Bruk

Klassen `ds-input` brukes på de fleste inndatafelter, som `<input>`, `<textarea>` og `<select>`.

```html
<input class="ds-input"/>
<textarea class="ds-input"/>
<select class="ds-input"/>
```

For et gyldig skjemaelement anbefaler vi å bruke `<ds-field>` sammen med en `<label>`.

```html
<ds-field class="ds-field">
  <label class="ds-label" >Checkbox</label>
  <input class="ds-input">
</ds-field>
```
