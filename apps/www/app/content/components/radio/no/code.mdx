## Bruk

Bruk klassen `ds-input` på `<input>` sammen med `type="radio"`.

```html
<input
  class="ds-input"
  type="radio"
/>
```

For et gyldig form-element anbefaler vi å bruke `<ds-field>` sammen med en `<label>`.

```html
<ds-field>
  <label class="ds-label">Radio</label>
  <input
    class="ds-input"
    type="radio"
    value="value"
    name="name"
  >
</ds-field>
```

Radio er `ds-input` med `type="radio"` så variabler og data-attributter som gjelder for [input](/no/components/docs/input/overview) gjelder også for radio.

## React

I React er det enklere å bygge opp en radio, da vi har en ferdig komponent som tar seg av det meste av logikken og tilstandshåndteringen for oss.

`Radio` er en sammensatt komponent som bruker [field](/no/components/docs/field/overview), [label](/no/components/docs/label/overview), [validation message](/no/components/docs/validation-message/overview) og [input](/no/components/docs/input/overview) for å lage en radio.

I tillegg har vi en egen [useRadioGroup](/no/components/utilities/use-radio-group) React-hook for å lettere håndtere en gruppe av Radio-komponenter.

```tsx
import { Radio } from '@digdir/designsystemet-react';

<Radio label="Radio" value="value" name="name" />
```
