## Usage

The component is an `<a>` element, and on MDN Web Docs you can read more about the [anchor HTML tag (mozilla.org)](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/a).
Use the class name `ds-link` on an `<a>` element to apply the link style.

## CSS variables and data attributes

Sizes are controlled with [data-size](/en/fundamentals/code/sizes) and colors with [data-color](/en/fundamentals/code/colors). The component will inherit from the closest parent where these are set.
