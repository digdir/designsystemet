## Usage

The component is an `<a>` element, and on MDN Web Docs you can read more about the [anchor HTML tag (mozilla.org)](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/a).

```tsx
import { Link } from '@digdir/designsystemet-react';

<Link href='https://designsystemet.no/'>Go to the designsystemet</Link>;
```

## HTML

Use the class name `ds-link` on an `<a>` element to apply the style of the `Link` component.

```html
<a href="https://designsystemet.no/" class="ds-link">Go to the design system</a>
```
