## Bruk

Komponenten er et `<a>`-element, og på MDN Web Docs kan du lese mer om [anchor HTML-taggen (mozilla.org)](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/a).

```tsx
import { Link } from '@digdir/designsystemet-react';

<Link href='https://designsystemet.no/'>Gå til designsystemet</Link>;
```

## HTML

Bruk klassenavnet `ds-link` på et `<a>`-element for å bruke stilen til `Link`-komponenten.

```html
<a href="https://designsystemet.no/" class="ds-link">Gå til designsystemet</a>
```
