## Bruk
```tsx
/* Utan ramme */
import { Details } from '@digdir/designsystemet-react';

<Details>
  <Details.Summary>Details heading text</Details.Summary>
  <Details.Content>Details content</Details.Content>
</Details>

/* Med ramme */

import { Details, Card } from '@digdir/designsystemet-react';

<Card>
  <Details>
    <Details.Summary>Details heading text</Details.Summary>
    <Details.Content>Details content</Details.Content>
  </Details>
</Card>
```

## Eksempler

### Ramme og variant

Dersom du legg `Details` som einaste barn av `Card` får du ramme rundt heile `Details`.
Du kan også bruke `variant="tinted"` på enten `Details` eller `Card` for å få ein lysare bakgrunn. 

<Story layout="block" story="InCardWithColor" />

### Kontrollert

`Details` held sjølv styr på om den er open eller lukka, men dette kan òg kontrollerast utanfrå.

<Story layout="block" story="Controlled" />

## HTML

`Details` er en native `<details>`-tagg.
Vi polyfyller ein feil i Firefox når den blir brukt saman med Android Talkback skjermlesar for å annonsere tilstand og rolle riktig.

```html
<!-- uten ramme -->
<details class="ds-details">
  <summary>
    Details heading text
  </summary>
  <div>
    Details content
  </div>
</details>

<!-- med ramme -->
<div class="ds-card">
  <details class="ds-details">
    <summary>
      Details heading text
    </summary>
    <div>
      Details content
    </div>
  </details>
</div>
```
