## Usage
```tsx
/* Without frame */
import { Details } from '@digdir/designsystemet-react';

<Details> 
<Details.Summary>Details heading text</Details.Summary> 
<Details.Content>Details content</Details.Content>
</Details>

/* With frame */

import { Details, Card } from '@digdir/designsystemet-react';

<Card> 
<Details> 
<Details.Summary>Details heading text</Details.Summary> 
<Details.Content>Details content</Details.Content> 
</Details>
</Card>
```

## Examples

### Frame and variant

If you put `Details` as the only child of `Card`, you get a frame around the whole `Details`.
You can also use `variant="tinted"` on either `Details` or `Card` to get a lighter background.

## HTML

`Details` is built on the web component [u-details (github.io)](https://u-elements.github.io/u-elements/elements/u-details), which you can use.
It is built functionally similar to the native element `<details>`, but with better accessibility.

You use the class name `.ds-details` on the main element.

You can choose not to have a nested `<div>` around the main content, but if you have this, it gets `margin`.

If you want to use tinted variant, add the attribute `data-variant="tinted"` to the main element or `.ds-card`.

```html
<!-- without frame -->
<u-details class="ds-details"> 
<u-summary> 
Details heading text 
</u-summary> 
<div> 
Details content 
</div>
</u-details>

<!-- with frame -->
<div class="ds-card"> 
<u-details class="ds-details"> 
<u-summary> 
Details heading text 
</u-summary> 
<div> 
Details content 
</div> 
</u-details>
</div>

<!-- without u-elements -->
<details class="ds-details"> 
<summary> 
Details heading text 
</summary> 
<div> 
Details content 
</div>
</details>
```
