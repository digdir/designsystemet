`Select` lets the user choose one option from a list.

<Story story="Preview" />

## Usage

```tsx
import { Select, Label } from '@digdir/designsystemet-react';
<>
<Label>Choose a mountain</Label>
<Select>
  <Select.Option value="everest">Mount Everest</Select.Option>
</Select>
</>
```
