## Use

```tsx
import { Select, Label } from '@digdir/designsystemet-react';

<>
  <Label>Select a mountain</Label>
  <Select>
    <Select.Option value="everest">Mount Everest</Select.Option>
  </Select>
</>
```

### ReadOnly

`<select>` is always `:read-only`, and does not support this in the same way as input types you can type into. If you set `aria-readonly` on a `<select>`, we ensure that it behaves as if it has read-only access, by preventing `onKeyDown` and `onMouseDown`. The user will still be able to focus on the element.

## HTML
Append the class name `ds-select` to `<select>`.

```html
<select class="ds-select">
  <option value="1">Option 1</option>
  <option value="2">Option 2</option>
  <option value="3">Option 3</option>
</select>
```

If you want `readonly` on a select, you must stop `onKeyDown` and `onMouseDown` yourself to prevent changing the state.
