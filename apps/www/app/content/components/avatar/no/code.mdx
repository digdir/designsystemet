## Bruk

Avatar bruker klassenavnet `ds-avatar` på en `<span>` med `role="img"`. Sørg for at avatar har [tilgjengelig navn](/no/components/docs/avatar/accessibility).

## CSS variabler og data-attributter
Størrelser styres med [`data-size`](/no/fundamentals/code/sizes) og farger med [`data-color`](/no/fundamentals/code/colors). Komponenten vil arve nærmeste forelder med disse satt.

## Eksempel

### Varianter

`variant` prop kan settes til `square` eller `circle`. `circle` er standard.

<Story story="ShapeVariants" />

### Med bilde eller ikon
Avatar viser som standard et brukerikon, men dette kan overstyres i CSS med variabelen `--dsc-avatar-icon-url`. Dersom du legger bilder eller ikon som barn vil disse få `aria-hidden="true"` automatisk i React for å unngå dobbel informasjon. Du må selv sette `aria-label` på avatar for å forklare hvem eller hva avataren representerer.

<Story story="WithImageAndIcon" />


## React

```tsx
import { Avatar } from '@digdir/designsystemet-react';

<>
  {/* Med standard brukerikon */}
  <Avatar aria-label="Ola Nordmann" />

  {/* Med initialer */}
  <Avatar aria-label="Ola Nordmann">ON</Avatar>
  {/* eller */}
  <Avatar aria-label="Ola Nordmann" data-initials="ON" />

  {/* Med bilde */}
  <Avatar aria-label="Ola Nordmann">
    <img src="..." />
  </Avatar>

  {/* Sammen med synlig tekst */}
  <Avatar aria-hidden>ON</Avatar>
  <span>Ola Nordmann</span>
</>
```
