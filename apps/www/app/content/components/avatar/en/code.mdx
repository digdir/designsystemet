## Usage

Avatar uses the class name `ds-avatar` on a `<span>` with `role="img"`.

```html
<!-- With default user icon -->
<span class="ds-avatar" role="img" aria-label="Ola Nordmann"></span>

<!-- With initials -->
<span class="ds-avatar" role="img" aria-label="Ola Nordmann">ON</span>
<!-- or -->
<span class="ds-avatar" role="img" aria-label="Ola Nordmann" data-initials="ON"></span>

<!-- With image -->
  <img src="..." aria-hidden="true" />
</span>

<!-- Together with visible text -->
<span class="ds-avatar" aria-hidden="true">ON</span>
<span>Ola Nordmann</span>
```

## Example

### Default Icon

If you do not send any `children`, the avatar will display a default user icon.

This can be overridden in CSS with the `--dsc-avatar-icon-url` variable. See the list of CSS variables at the bottom of the page.

<Story story="NoName" />

### Variants

The `variant` prop can be set to `square` or `circle`. `circle` is the default.

<Story story="ShapeVariants" />

### With image or icon

If you add images or icons as children, they will automatically get `aria-hidden="true"` to avoid duplicate information.

You must set `aria-label` on the avatar yourself to explain who or what the avatar represents.

## React

```tsx
import { Avatar } from '@digdir/designsystemet-react';

<>
  {/* With default user icon */}
  <Avatar aria-label="Ola Nordmann" />

  {/* With initials */}
  <Avatar aria-label="Ola Nordmann">ON</Avatar>
  {/* or */}
  <Avatar aria-label="Ola Nordmann" data-initials="ON" />

  {/* With image */}
  <Avatar aria-label="Ola Nordmann">
    <img src="..." />
  </Avatar>

  {/* Together with visible text */}
  <Avatar aria-hidden>ON</Avatar>
  <span>Ola Nordmann</span>
</>
```
