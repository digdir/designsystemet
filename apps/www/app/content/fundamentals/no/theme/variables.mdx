Design Tokens er fleksible variabler som kan benyttes uavhengig av teknologi eller designverktøy. Denne oversikten viser alle variablene som blir generert av [Designsystemet sitt kommandolinjeverktøy](/no/fundamentals/code/cli-config).

## Farger

Fargevariabler som brukes til å definere fargepaletten i temaet. Bruk attributtet `data-color` for å endre farge eller `data-color-scheme` for å endre fargescheme på HTML-elementer.

<ColorTokensTable />

## Størrelser

Størrelsevariabler som brukes til å definere avstander og størrelser. Disse variablene tilpasser seg basert på størrelsesmodusen satt ved bruk av `data-size` attributtet eller variablene `--ds-size-base` og `--ds-size-step`.

<SizeVariablesTable withPreview />

## Border-radius

Border radius variabler som brukes til å definere border radius på komponenter og former. Bruk variablene `--ds-border-radius-base` og `--ds-border-radius-step` for å justere intensiteten på border radius.

<BorderRadiusVariablesTable withPreview />

## Skygger

Skyggevariabler som brukes til å definere skygger på komponenter og elementer.

<ShadowVariablesTable withPreview />

## Opasitet

Opasitetsvariabler som brukes til å definere opasitetsnivåer på komponenter og elementer.

<OpacityVariablesTable withPreview />

## Typografi

Typografivariabler som brukes til å definere skriftstiler, størrelser og vekter.

<TypographyVariablesTable />
