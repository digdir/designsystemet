Designsystemet er laget for å kunne brukes sammen med din egen visuelle profil - inkludert farger og andre preferanser. Du trenger dermed ikke å bygge et nytt designsystem fra bunnen av. Med [Designsystemets Temabygger](https://theme.designsystemet.no) kan du sette opp et eget tema og generere Design Tokens som kan brukes både i designverktøy og i kode.

<Alert data-color="info">
Vi anbefaler å ha en kobling mellom design og kode, slik at designere og utviklere arbeider etter de samme reglene. 
Derfor er det viktig at dere er både en utvikler og en designer med i prosessen når dere setter opp deres eget tema.
</Alert>


## Slik går du frem:

1. Kopier config-fila under, og lagre den som `designsystemet.config.json`

```json
{
  "outDir": "./design-tokens",
  "clean": "true",
  "themes": {
    "theme": {
      "colors": {
        "main": { "primary": "#0062BA", "accent": "#1E98F5" },
        "neutral": "#1E2B3C",
        "support": { "extra1": "#F45F63", "extra2": "#E5AA20" }
      },
      "borderRadius": 4
    }
  }
}
```

2. Opprett et Git repository du kan bruke for å lagre Design Tokens. Legg `designsystemet.config.json`-fila i roten av repoet.

3. Klon repoet lokalt på din maskin, åpne en terminal og naviger til mappen med repoet.

4. Pass på at du har node.js installert med en versjon høyere enn 20.

5. Kjør denne kommandoen i terminalen:
```bash
npx @digdir/designsystemet@latest tokens create --config designsystemet.config.json
```
Dette vil generere Design Tokens basert på config-fila og lagre dem i en mappe som heter `design-tokens`.

6. Nå er Design Tokens generert og du kan pushe endringene til ditt repo.

<Alert data-color="info">
Nå skal vi inn i Figma. Pass på at du bruker desktop-appen til Figma, ikke nettleserversjonen.
</Alert>

7. Hent komponentbiblioteket fra [Figma Community (figma.com)](https://www.figma.com/community/file/1322138390374166141/designsystemet-core-ui-kit) (Trykk "Open in Figma") til din organisasjon som har mimimum pro-lisens. Merk at dette blir en kopi av komponentbiblioteket uten noen kobling mot hovedfilen. 

8. Installer pluginen [Tokens Studio (figma.com)](https://www.figma.com/community/plugin/843461159747178978/Tokens-Studio-for-Figma-(Figma-Tokens)) for Figma. Tokens Studio sørger for kobling mellom Figma og kode. 

<Image
  src='/img/tokenstudio.png'
  alt='Skjermbilde av Figma som vise hvor du finner plugins'
  boxShadow={false}
/>

  - Følg Tokens Studio sin guide "[Sync your Design Tokens with code (tokens.studio)](https://docs.tokens.studio/token-storage-and-sync/sync-provider-overview)" for å sette opp kobling til ditt Git repo. 
  - Husk å sette "Token storage location" til `design-tokens` (eller det du har valgt å kalle mappen din med Design Tokens).

<Alert data-color="info">
Vi anbefaler å *ikke* pushe endringer i Design Tokens via Tokens Studio, da dette kan føre til at du miste overstyringene dine når du oppdatererer.
</Alert>

9. Gå til [Temabyggeren](https://theme.designsystemet.no) og generer fargeskalaer ut fra dine profilfarger. Klikk "Ta i bruk tema". Kopier config-filen og lim inn i `designsystemet.config.json`-fila i ditt repo. Kjør kommandoen under og push endringene.

```bash
npx @digdir/designsystemet@latest tokens create --config designsystemet.config.json
```
**Husk at du må kjøre denne kommandoen hver gang du gjør endringer i config-filen.**


10. I Tokens Studio velg "Pull"-ikonet for å hente de nye tokenene inn i Figma.

<Image
  src='/img/fundamentals/themebuilder/own-theme/tokens-studio-pull-button.png'
  alt='Skjermbilde av Tokens Studio og hvor du finner pull knappen'
  boxShadow={false}
/>

11. Velg `Styles & variables` og `Export styles & variables to Figma`. 

<Image
  src='/img/fundamentals/themebuilder/own-theme/tokens-export.png'
  alt='Skjermbilde av Tokens Studio og hvor du finner export knappen'
  boxShadow={false}
/>

Pass på at du har disse innstillingene når du eksporterer variablene: 

<Image
  src='/img/fundamentals/themebuilder/own-theme/tokens-export-options.png'
  alt='Skjermbilde av Tokens Studio og hvilke innstillinger du skal bruke når du eksporterer tokens til Figma'
  boxShadow={false}
/>

Velg "Themes" og deretter "Select All":

<Image
  src='/img/fundamentals/themebuilder/own-theme/tokens-export-themes-select.png'
  alt='Skjermbilde av Tokens Studio som viser at du må velge Themes og deretter Select All'
  boxShadow={false}
/>

12. Nå skal du se alle komponentene med dine egne profilfarger i Figma. Når Design Tokens oppdateres i koden, kan du enkelt hente de nye verdiene inn i Figma ved å bruke "Pull"-knappen i Tokens Studio igjen.
