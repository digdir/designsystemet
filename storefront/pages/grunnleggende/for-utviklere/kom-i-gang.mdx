## Kom i gang med CSS

For å bruke designsystemet i kode må du installere `@digdir/designsystem-css` fra npm.

```bash
npm install @digdir/designsystem-css
```

Du kan også bruke yarn:

```bash
yarn add @digdir/designsystem-css
```

Når du har installert pakken må du importere `index.css` prosjektet ditt:

```js
import '@digdir/designsystem-css/index.css';
```

Denne inneholder alle stilene du trenger for å bruke designsystemet. Du kan også importere enkelte deler av designsystemet, for eksempel:

```js
/* Utilities må bli importert dersom du ikke importerer index.css */
import '@digdir/designsystem-css/utilities.css';
import '@digdir/designsystem-css/button.css';
```
